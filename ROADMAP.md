# Roadmap (2023)

Цели Server Monitoring (FastAPI + psutil + Prometheus), разбиты по кварталам 2023.

Q1 2023
- Базовые эндпоинты: `/health`, `/metrics/json`, `/metrics`.
- Снимок метрик: CPU/RAM/Disk/Net.
- Минимальная конфигурация и запуск через `uvicorn`.

Q2 2023
- Добавить системные метрики: загрузка дисков по устройствам, процессы‑топ.
- Экспорт дополнительных метрик: файловые дескрипторы, uptime.
- Тесты: `pytest` для сборщиков метрик и эндпоинтов.

Q3 2023
- CI: GitHub Actions (Python 3.13), линтеры (`ruff`, `black`, `mypy`).
- Наблюдаемость: структурированное логирование, `/metrics` улучшения.
- Конфигурация: `pydantic-settings`, параметры интервалов и фильтров.

Q4 2023
- Интеграция с Prometheus/Grafana: пример dashboard, compose.
- Безопасность: ограничения доступа, rate limiting, CORS.
- Производительность: профилирование, оптимизация сбора метрик.

Технические заметки
- Использовать `psutil` для кроссплатформенного сбора.
- Метрики именовать в стиле Prometheus, единицы измерения в названиях.
- Сервис без состояния, масштабирование горизонтально.
